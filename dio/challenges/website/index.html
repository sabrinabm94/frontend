Textos e Marcação

HTML5 é a versão mais recente da linguagem de marcação para páginas web, que oferece novas funcionalidades e tags, como destacar palavras e elementos semânticos como <header> e <footer>.

Você pode italizar ou sublinhar partes do texto para dar ênfase. Algumas tags como HTML ajudam a abreviar termos, enquanto diminuem o tamanho da fonte para notas de rodapé.
